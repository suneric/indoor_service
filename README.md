# Indoor Service Robot
This is a simulation of a mobile robot for performing indoor service, including autonomous navigation, self-charging, and door opening.

## prerequisite
- Linux: Ubuntu 20.04
- ROS: Noetic
- OpenCV
- RealSense SDK
- tensorflow
- pytorch
  - pandas
  - tqdm
  - seaborn

## packages
1. ids_gazebo
2. ids_description
3. ids_control
4. ids_detection
5. ids_joy
6. ids_learning
7. ids_task
